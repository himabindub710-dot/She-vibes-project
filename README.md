#  5 Min Story Korner
## Description 

WHAT IS STORY KORNER?

Every child deserves a great story.

Story Korner is a storytelling app designed with love for young minds. Packed with enchanting tales, playful characters, and vivid illustrations, it fits perfectly into busy family routines — because every great bedtime, naptime, or quiet moment deserves a little magic that lasts exactly five minutes.

## Installation & Setup
No need of any login credentials or setup required. 
Click on the story name and it navigates to the story, you either read the story or listen to the story audio.
